# xrwvm-fullstack_developer_capstone

## Project Name: fullstack_developer_capstone

A full-stack Django/React car dealership application built as part of the IBM Full Stack Developer Capstone project.

## Repository
- Repository name: capstone-fullstack
- Project name: fullstack_developer_capstone

## Features
- Browse car dealerships
- View dealer details and reviews
- Post reviews for dealerships
- Sentiment analysis on reviews
- User authentication
